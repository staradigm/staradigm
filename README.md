## 🚀 Projects

### Japan Travel Assistant CLI

**Description:** A command‑line tool that reads a simple markdown itinerary, resolves station names to Tokyo Metro lines, and outputs step‑by‑step subway directions for each leg of the trip.

**Highlights**
- Parses markdown itineraries with flexible date/location formats
- Uses a static station‑to‑line map (CSV) for quick lookups
- Generates human‑readable route instructions with transfer points
- Includes basic tests and a GitHub Actions workflow for linting

**Tech Stack:** `Python` `Click` `Pandas` `GitHub Actions`

[Repository](https://github.com/staradigm/japan-travel-assistant) · [Live Demo](https://github.com/staradigm/japan-travel-assistant#readme)

### Open‑Source Contribution: Travel‑Data‑Utils

**Description:** A small utility library for normalizing and validating travel‑related data (IATA codes, station codes, time‑zone conversions).

**Highlights**
- Provides pure‑Python functions with comprehensive type hints
- Includes 150+ unit tests covering edge cases
- Published to PyPI as `travel-data-utils`
- Used internally in several personal projects

**Tech Stack:** `Python` `Poetry` `PyPI`
